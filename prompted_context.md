# Prompted.sbs - Context

## Project Overview
Personal portfolio website for prompted.sbs showcasing AI expertise and projects. Multi-language support (Dutch/NL, English/EN, French/FR).

## Location
**Local:** `~/ai/prompted-website/`
**Deployed:** NAS Cloudflare Tunnel via prompted.sbs domain

## Tech Stack
- Static HTML/CSS/JavaScript
- No build process - direct file editing
- Responsive design with mobile optimizations

## File Structure
```
~/ai/prompted-website/
├── index.html          # Dutch (NL) - primary language
├── en.html            # English version
├── fr.html            # French version
├── styles.css         # Shared stylesheet
├── script.js          # Shared JavaScript
├── favicon.ico        # Robot head favicon (fills full space, no antenna)
├── favicon.png        # High-res PNG backup
├── favicon.svg        # Vector version
├── logo.png           # Prompted brain logo (detailed, for header/branding)
└── robots.txt         # SEO configuration
```

## Branding Assets

### Favicon (favicon.ico)
- **Design:** Robot head without antenna, fills entire favicon space
- **Format:** ICO (multi-resolution)
- **Note:** Intentionally different from logo - favicon optimized for small size clarity

### Logo (logo.png)
- **Design:** Detailed brain/AI logo generated by Gemini
- **Usage:** Website header, branding elements
- **Format:** PNG (high resolution)

**Design Philosophy:** Logo and favicon don't need to match - logo provides detail for large display, favicon provides clarity at 16x16px.

- **Remote:** Private GitHub repository
